Learning Path

Welcome to Learnig Path. We created this site for those who want to learn web development (programming in general) and has no clue on how to get started.

Each learning path is created with following objectives in mind:

  • To provide smooth learning curve and easier entry to get started
  • To povide essentials skills required for a typical (web) developer
  • To only focus on topics that are essentiail (or need to know now)
  • To help you with the problem of "You don't know what you don't know"

Contribution

We would definitely love that. Please fork the repository and submit pull request when you are ready. This site is hosted with Github Pages and deploy directly from gh-pages branch.

The source code of this website only use basic HTML/CSS generated by Github. So if you want to play along with it, this is a good starting point too.
